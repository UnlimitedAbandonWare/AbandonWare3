# Contributing

Thanks for your interest in contributing!

## Pull Requests
- Fork the repo and create a branch from `main`.
- Ensure tests pass and code is formatted.
- Do not commit secrets or environment-specific configs.

## Development
- Configure runtime secrets via environment variables.
- See `main/resources/*.example` for required variables.

## Code of Conduct
Be respectful and constructive. We follow the Contributor Covenant.
